## Installation

1. Download `ScreenRuler_Pro_v1.0.0.exe`
2. Run the executable (no installation required)
3. The application starts in Edit mode - press **P** to toggle to Work mode

## Keyboard Shortcuts

- **P** - Toggle Edit/Work Mode
- **M** - Cycle Measurement Mode
- **U** - Cycle Units
- **T** - Cycle Themes
- **C** - Copy Measurement
- **R** - Reset Position
- **G** - Toggle Guides
- **V** - Toggle Labels
- **L** - Cycle Lock (None/Horizontal/Vertical)
- **[** / **]** - Adjust Fractions/Polygon Sides
- **,** / **.** - Adjust Thickness
- **+** / **-** - Adjust Opacity
- **Space** - Minimize to Tray
- **Esc** - Exit
